# 🧠 KinKeeper

KinKeeper is a modern family & relationship activity tracker built with React.js.
It allows users to manage and visualize daily interactions like calls, texts, and video activities in a simple timeline interface.

## 🚀 Live Demo

🔗 https://kin-keeper-beta.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/omarfaruq2806/KinKeeper

## ✨ Features

📅 Interactive activity timeline
📞 Filter activities by Call, Text, and Video
🔄 Dynamic state-based UI updates
⚡ Fast and responsive React interface
🎯 Simple and clean user experience
📱 Mobile-friendly design


## 🛠️ Tech Stack

React.js
JavaScript (ES6+)
HTML5
Tailwind CSS
React Context API
React Hooks
React Icons
React Spinners
Vercel (Deployment)
